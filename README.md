## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/NullShine69/MY-Portfolio.git
cd My-Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```
